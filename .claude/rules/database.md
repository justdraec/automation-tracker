- The Supabase client is in `src/lib/supabase.ts`. Never instantiate a new client inline in a component.
- Always handle the `error` returned from Supabase: `const { data, error } = await supabase...; if (error) throw error`
- Never use `select('*')` тАФ always specify only the columns needed.
- The `id` column is auto-generated by Supabase. Never set `id: Date.now()` in insert payloads тАФ omit `id` entirely and let Supabase assign it.
- JSONB columns: `process_steps`, `auth_data`, `mappings`, `dependencies`, `tool_chips` тАФ stored as JSONB arrays.
- The `status` field must be one of: `pending-review`, `not-started`, `in-progress`, `built`, `on-hold`.
- Edge functions run on Deno. Use `Deno.env.get('KEY_NAME')` not `process.env`.
- Table names are `snake_case` plural. Column names are `snake_case`. Function names in `supabase.ts` are `camelCase` verb-first.
